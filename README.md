# Planarity Checker

A web platform where users can draw a graph and check whether it is planar or non-planar.

## Version 1 goals
- Add vertices
- Add edges
- Check planarity
- Show result clearly

## Planned stack
- Frontend: React
- Backend: FastAPI
- Graph algorithm: NetworkX

## Future ideas
- Planar embedding
- Euler formula explanation
- Highlight K5 / K3,3 patterns
- Save and share graphs
